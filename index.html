Hvacpi

HVACpi is an open source project that supports the use of the Raspberry Pi board computer to run HVAC systems.

Download this project as a .zip file Download this project as a tar.gz file

The primary goal of the HVACpi project is to create a low cost web enabled home heating and cooling management system which consists of highly available technology. HVACpi will use the raspberry pi, RPi GPIO, raspbian linux, lamp, and python.
